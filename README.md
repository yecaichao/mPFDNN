# mPFDNN
Material-Property-Field-based Deep Neural Network (mPFDNN), a Forward Analytical Material-Specific DNN framework (FAMS-DNN) integrate Material Property Fields (MPF) with the Hopfield network architecture.

# Installation
conda create -n FAMS-DNN python=3.10  
conda activate FAMS-DNN  
cd FAMS-DNN  
pip install -r requirements.txt  
pip setup.py install
